I will tell you how to use it!!!

in only two steps

Import code from cdn or take main.js and add it into your project
Code: html
< script  type= "text/javascript"  src= "https://cdn.peixweb.com/js/myscroll.js" > < /script >

To customize an scroll bar simply type MyScroll.run inside script tag in your html document. run function receive an options object, you will need to fill it with desired values.

Code: javascript
MyScroll.run({
selector: '#your-scrollable-container',
scrollWidth: 5 ,
scrollTrackColor: '#f5f5f6' ,
scrollThumbColor: '#4f5b62',
scrollThumbHoverColor: '#000a12'
});

The selector property is simply the css selector, if you want to style a tag wid id use: selector:'#my-component', if you want to style a class use dot, for example: selector:'.my-class' or if you want to style a standard html components use for example: selector:'body'.
