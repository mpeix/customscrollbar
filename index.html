MyScroll

Discover MyScroll, an easy to use tool to customize your scrollbars

Here, you will find a demo tool with sample snippets to make your integration with MyScroll fast and easy.

You can strart using MyScroll in only two steps

Import code from cdn or take myscroll.js file and add it into your project

Code: html
Copy to clipboard
< script  src= "https://cdn.peixweb.com/js/myscroll.js" > < /script >

To customize an scroll bar simply type MyScroll.run inside script tag in your html document. run function receive an options object, you will need to fill it with desired values.

Code: javascript
Copy to clipboard
MyScroll.run({
selector: '#scrollable-container',
scrollWidth: 5 ,
scrollTrackColor: '#f5f5f6' ,
scrollThumbColor: '#4f5b62',
scrollThumbHoverColor: '#000a12' ,
firefoxWidth: 'auto' ,
useArrows: false
});

The selector property is simply the css selector, if you want to style a tag with id use hash symbol: selector:' #my-component', if you want to style a class use dot, for example: selector: '.my-class' or if you want to style a standard tag use tag name directly, for example: selector: 'body'.
